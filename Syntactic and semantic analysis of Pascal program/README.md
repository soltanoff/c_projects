# Syntactic and semantic analysis of Pascal program
========================

О ПРОГРАММЕ
-----------

Программа, осуществляя анализ текста программы на языке Pascal, сообщает 
о правильности текста программы как синтаксически, так и семантически. В 
противном случае сообщает о выявленных семантических и синтаксических 
ошибках в тексте программы.

Язык Pascal – структурированный, компилируемый язык, который один из 
наиболее известных языков программирования, используется для обучения 
программированию в старших классах и на первых курсах вузов, является 
базой для ряда других языков. 

Программа анализирует подмножество языка Pascal, содержащее следующее:
	
	•	Инициализация переменной либо числовых, либо строковых типов данных
	
	•	Блок begin-end
	
	•	Математические выражения
	
	•	Логические выражения
	
	•	Условный оператор if-then-else
	
	•	Присвоение строк

	(!) Алгоритмы и структуры данных, руководство пользователя, результаты тестирования можно лицезреть в пояснительной записке.
Курсовой проект по дисциплине "Теория языков программирования и методы трансляции". 
Написан на языке СИ. 

БИНАРИ
-----------

Для демонстрации работы программы прикладываю бинарь программы и 
необходимые файлы для запуска.

НЕОБХОДИМЫЕ ФАЙЛЫ И СПОСОБЫ ЗАПУСКА
-----------------

Имена необходимых файлов по умолчанию:

	1. property_table.db - файл с таблицей грамматики свойств.

	2. input.txt - файл с текстом программы на языке Pascal.
	
Способы запуска:

	1. Передача в качестве параметров имена необходимых файлов: 
	"binary.exe" input.txt property_table.db
	
	2. Передача 1-одного параметра (имя входного файла с текстом программы):
	"binary.exe" input.txt
	
	3. Стандартный запуск: "binary.exe"
	
	* При выполнении пунктов 2-3 необходимые файлы должны присутствовать 
	в директории бинарного файла и должны иметь имена по умолчанию.

КАК СОБРАТЬ ПРОЕКТ
------------------

	1. Сгенерируйте исходный код парсера при помощи утилит bison и flex.
	
	2. Выполните вставку фрагмента из файла "PUT THIS TO bison.txt.tab.h" 
	в соотвествующее место.
	
	* Пункт 1-2 можно не делать, ибо файлы сгенерированы
	
	3. Заголовочные файлы из include и bison.txt.tab.h вложите по своим местам.
	
	5. Расположите в корне проекта файл property_table.db и создайте файл input.txt, 
	в котором будет находиться текст программы на языке Pascal.
	
	4. lex.yy.с - файл ресурсов, и bison.txt.tab.с - файл исходного кода
	
DEBUG-ВЫВОД
-----------

Для активации сие вывода опишите следующие дерективы в файле исходного кода:

"#define DEBUG_T" - вывод результата семантического анализа текста программы.

"#define DEBUG_TP" - вывод таблиц грамматики свойств.
